WLS2配置代理

代理软件设置

  1. 以 Clash 为例，选择主页选项卡，开启允许局域网。
  2. 打开 Windows Defender 防火墙，允许应用或功能通过 Windows Defender 防火墙。点击更改设置，找到 Clash for Windows，然后勾选专用和公用。
    如果找不到 Clash for Windows，点击下方的允许其他应用，然后找到安装路径，将 Clash for Windows.exe 文件加入进来。
